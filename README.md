# STM32 Temperature Sensor Project

Проект на базе микроконтроллера NUCLEO-L053R8 для считывания температуры с цифрового датчика и вывода результата через UART.

## Описание проекта

Данный проект реализует работу микроконтроллера STM32 с температурным датчиком.  
Микроконтроллер получает данные с датчика температуры, обрабатывает их и передаёт результат в терминал компьютера через UART.

Проект может использоваться как учебный пример работы с:
- GPIO;
- таймерами;
- интерфейсом 1-Wire;
- UART;
- внешним цифровым датчиком температуры;
- STM32CubeIDE и HAL-библиотекой.

## Используемые компоненты

- Отладочная плата STM32 NUCLEO-L053R8;
- Датчик температуры DS18B20;
- Соединительные провода;
- Резистор подтяжки для линии данных датчика;
- ПК с установленной STM32CubeIDE;
- Терминальная программа для просмотра UART-вывода.

## Основной функционал

Проект выполняет следующие действия:

1. Инициализирует периферию микроконтроллера.
2. Настраивает GPIO для работы с датчиком температуры.
3. Выполняет обмен данными с датчиком по интерфейсу 1-Wire.
4. Считывает значение температуры.
5. Преобразует полученные данные в удобный формат.
6. Отправляет результат измерения через UART.
